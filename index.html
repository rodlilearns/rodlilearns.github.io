Create a psql container using Podman

  1. Login to a container registry > $ podman login registry.redhat.io
  2. Create a container using podman run > $ podman run -d --name container_name -e POSTGRESQL_USER=username -e POSTGRESQL_PASSWORD=password -e POSTGRESQL_DATABASE=database_name -p 5432:5432 rhel8/postgresql-13
  3. Verify container exists
